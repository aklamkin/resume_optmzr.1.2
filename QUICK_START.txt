🎯 DEPLOY ON SYNOLOGY NAS (3 STEPS)
====================================

Step 1: Edit Configuration
--------------------------
File: .env
Change this line:
    REACT_APP_BACKEND_URL=http://192.168.1.100:8080
Replace 192.168.1.100 with YOUR NAS IP address

Step 2: Upload to NAS
----------------------
Method A (GUI):
  1. Open DSM → File Station
  2. Upload entire folder to /docker/resumeai/
  
Method B (SSH):
  scp -r . admin@YOUR_NAS_IP:/volume1/docker/resumeai/

Step 3: Deploy
--------------
Method A (Container Manager):
  1. Open Container Manager
  2. Project → Create
  3. Name: resumeai
  4. Path: /docker/resumeai
  5. Source: docker-compose.yml
  6. Done!

Method B (SSH):
  cd /volume1/docker/resumeai
  sudo docker compose up -d

🆘 TROUBLESHOOTING
==================
Problem: Can't access on port 8080
Solution: Check firewall, verify port not in use

Problem: AI analysis fails  
Solution: Verify GEMINI_API_KEY in .env file

Problem: Container won't start
Solution: Check logs with: docker compose logs [container_name]

Problem: Database connection error
Solution: docker compose restart mongodb
